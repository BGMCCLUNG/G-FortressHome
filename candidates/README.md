# Candidate Properties

One subfolder per property researched. Use the TEMPLATE.md as your starting point.

## Naming Convention
`[street-number]-[street-name]-[neighborhood]/`

Example: `1248-liverpool-st-manchester/`

## Status Key
- 🔍 Research — gathering initial data
- 👀 Watch — viable, monitoring
- ✅ Pursue — actively working toward acquisition
- ❌ Pass — disqualified (note reason)
